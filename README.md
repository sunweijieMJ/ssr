# LaneHub  user ssr

> vue2.x vuex koa

## Build Setup
Requires Node.js 8+

``` bash
# install dependencies
yarn install 

# serve in dev mode, with hot reload at localhost:3000
yarn run dev

# build for production
yarn run build

# serve in production mode
yarn start
```
